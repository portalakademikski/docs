### Apa Itu Tugas Kuliah?

Menu **Tugas Kuliah** merupakan fitur pada Virtual Class yang digunakan untuk menampilkan daftar tugas yang diberikan oleh dosen pada mata kuliah yang diikuti. Melalui menu ini, mahasiswa dapat melihat detail tugas, batas waktu pengumpulan, serta melakukan pengunggahan jawaban atau file tugas sesuai ketentuan.

### Tata Cara Mengakses Tugas Kuliah

<AccordionGroup>
  <Accordion title="Login pada Portal Akademik menggunakan akun Anda" icon="">
    Masukkan username dan password pada halaman login, kemudian tekan tombol **Login** untuk mengakses fitur-fitur di Portal Akademik.
  </Accordion>
  <Accordion title="Pilih menu Virtual Class > Tugas Kuliah">
    Sistem akan menampilkan daftar tugas kuliah yang tersedia/diupload oleh Dosen.

    ![Tugas Kuliah Daftar Tugas Kuliah](/images/TugasKuliah.DaftarTugasKuliah.png)
  </Accordion>
  <Accordion title="Tekan tombol File Soal, untuk mengunduh soal">
    Akan terunduh dokumen soal yang dilampirkan oleh dosen. Selain itu pada card terlihat informasi terkait tugas kuliah, seperti: Mata Kuliah, Waktu Awal dan Akhir, Nilai dan Catatan.

    ![Tugas Kuliah Download Soal](/images/TugasKuliah.DownloadSoal.png)
  </Accordion>
  <Accordion title="Tekan tombol File Jawaban, untuk upload upload jawaban">
    Setelah Anda melakukan klik pada **File Jawaban,** akan tampil data detail tugas kuliah. Pada halaman ini Anda dapat upload jawaban dengan klik**Upload File**  dengan mengubah ekstensi lembar jawaban menjadi PDF, JPG, RAR atau ZIP.

    ![Tugas Kuliah File Jawaban](/images/TugasKuliah.FileJawaban.png)

    ![Tugas Kuliah Upload File](/images/TugasKuliah.UploadFile.png)

    ![Tugas Kuliah Upload File2](/images/TugasKuliah.UploadFile2.png)
  </Accordion>
  <Accordion title="Tekan Simpan, jika sudah Anda lengkapi Jawaban">
    Untuk menyimpan dan mengirimkan tugas Anda bisa klik Simpan.

    ![Tugas Kuliah Simpan Upload File](/images/TugasKuliah.SimpanUploadFile.png)
  </Accordion>
</AccordionGroup>
