### Apa Itu Menu Referensi?

Menu **Referensi** merupakan fitur pada Virtual Class yang digunakan untuk menyediakan bahan bacaan atau sumber referensi pendukung pembelajaran. Melalui menu ini, mahasiswa dapat mengakses daftar referensi yang direkomendasikan oleh dosen untuk menunjang pemahaman materi perkuliahan.

### Cara Mengakses Referensi

1. **Login pada Portal Akademik menggunakan akun Anda**

   Masukkan username dan password pada halaman login, kemudian tekan tombol **Login** untuk mengakses fitur-fitur di Portal Akademik.
2. **Pilih menu Virtual Class \> Referensi**

   Sistem akan menampilkan akan menampilkan daftar file yang tersedia, Anda juga dapat melakukan pencarian materi menggunakan fitur search kata kunci.
