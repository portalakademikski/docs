### Apa Itu Menu File Sharing?

Menu **File Sharing** merupakan fitur pada Virtual Class yang digunakan untuk berbagi dan mengakses file pendukung perkuliahan. Melalui menu ini, mahasiswa dapat mengunduh file yang dibagikan oleh dosen atau mengunggah file sesuai dengan ketentuan yang berlaku pada kelas tersebut.

### Cara Mengakses dan Menambah File Sharing

1. **Login pada Portal Akademik menggunakan akun Anda**

   Masukkan username dan password pada halaman login, kemudian tekan tombol **Login** untuk mengakses fitur-fitur di Portal Akademik.
2. **Pilih menu Virtual Class \> File Sharing**

   Sistem akan menampilkan akan menampilkan daftar file yang tersedia, Anda juga dapat melakukan download file dengan klik **Buka File**.

   </Frame>

3. **Tekan Upload File**

   Fitur ini dapat anda gunakan untuk melakukan Upload File Baru untuk kemudian di share ke Civitas Akademika pengguna Portal Akademik.
