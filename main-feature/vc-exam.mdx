### Apa Itu Menu Ujian Kelas?

Menu **Ujian Kelas** merupakan fitur pada Virtual Class yang digunakan untuk menampilkan daftar ujian yang diikuti oleh mahasiswa pada mata kuliah tertentu. Melalui menu ini, mahasiswa dapat melihat informasi jadwal ujian, durasi pengerjaan, serta mengerjakan ujian secara daring sesuai ketentuan yang berlaku.

<Info>
  Sub-menu Ujian Kelas dapat digunakan untuk pelaksanaan UAT dan UAS, dimana mahasiswa dapat mengunduh soal dan mengunggah jawaban serta mahasiswa dapat melihat nilai.
</Info>

### Langkah Mengakses dan Mengerjakan Ujian

<AccordionGroup>
  <Accordion title="Login pada Portal Akademik menggunakan akun Anda" icon="">
    Masukkan username dan password pada halaman login, kemudian tekan tombol\*\*Login \*\* untuk mengakses fitur-fitur di Portal Akademik.
  </Accordion>
  <Accordion title="Pilih menu Virtual Class > Ujian Kelas" icon="">
    Portal Akademik akan menampilkan daftar ujian yang tersedia.
  </Accordion>
  <Accordion title="Tekan tombol File Soal, untuk mengunduh soal">
    Akan terunduh dokumen soal yang dilampirkan oleh dosen. Selain itu pada card terlihat informasi, seperti: Mata Kuliah, Waktu Awal dan Akhir, Nilai dan Catatan.
  </Accordion>
  <Accordion title="Tekan tombol File Jawaban, untuk upload upload jawaban">
    Setelah Anda melakukan klik pada File Jawaban, akan tampil data detail tugas kuliah. Pada halaman ini Anda dapat upload jawaban dengan klik **Upload File**  dengan mengubah ekstensi lembar jawaban menjadi PDF, JPG, RAR atau ZIP.
  </Accordion>
  <Accordion title="Tekan Simpan, jika sudah Anda lengkapi Jawaban">
    Proses simpan untuk mengakhiri dan mengumpulkan jawaban.
  </Accordion>
</AccordionGroup>
