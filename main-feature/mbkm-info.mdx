title: "Informasi MBKM"
---

Menu **Informasi MBKM** merupakan fitur pada menu **Kampus Merdeka** yang digunakan untuk menampilkan informasi terkait program Merdeka Belajar Kampus Merdeka (MBKM). Melalui menu ini, mahasiswa dapat memperoleh informasi mengenai jenis program MBKM, persyaratan, jadwal pelaksanaan, serta ketentuan yang berlaku.

### Langkah Untuk Mengakses Informasi MBKM

1. **Login pada Portal Akademik menggunakan akun Anda**

   Masukkan username dan password pada halaman login, kemudian tekan tombol **Login** untuk mengakses fitur-fitur di Portal Akademik.
2. **Pilih menu Kampus Merdeka \> Informasi MBKM**

   Sistem akan menampilkan daftar informasi program MBKM yang tersedia dan di lengkapi fitur **Pencarian Data.**

2. **Download File Informasi**

   Untuk mengkases informasi MBKM secara legkap, Anda dapat klik **Download** pada list pengumuman yang ingin diunduh.
