<Info>
  Anda dapat **meninjau hasil evaluasi** berdasarkan semester, sesi tertentu dan grup, serta mengunduh laporan dalam format Excel jika diperlukan.
</Info>

### Melihat Data Hasil Kuesioner

1. Login pada Portal Akademik menggunakan akun Anda.
1. Klik menu **Informasi Akademik** pada navigasi menu.
2. Kemudian klik sub menu **Hasil Kuesioner**, penampilan data sesuai dengan dengan filter yang dipilih.
   - Filter Semester: Untuk memilih semester tertentu yang ingin Anda lihat datanya
   - Filter Sesi: Untuk memilih periode atau waktu pelaksanaan (misalnya sesi 1, sesi 2, atau periode khusus)
   - Filter Grup: Untuk memilih kelompok atau kategori kuesioner tertentu, seperti grup Matakuliah atau Grup Global sehingga tampilan data lebih terfokus.
3. Untuk mencetak laporan atau hasil kuesioner, klik tombol **Download XLSX**.
